# 🍽️ Zomato Data Analysis (EDA Project)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the Zomato restaurant dataset to uncover meaningful insights about restaurant trends, customer preferences, pricing, ratings, and online ordering behavior.

The analysis helps understand the factors affecting restaurant popularity and customer satisfaction using Python-based data analysis and visualization.

---

## 🎯 Problem Statement
Zomato collects large-scale restaurant data including ratings, votes, cost, and ordering facilities.  
The objective of this project is to analyze this dataset and extract useful business insights that can help restaurants improve services and help customers make better decisions.

---

## 📂 Dataset Information
The dataset contains restaurant-related information such as:

- Restaurant Name
- Location
- Average Cost for Two
- Online Order Availability
- Table Booking Availability
- Ratings
- Number of Votes
- Restaurant Type

## 🔍 Key Insights
- Most restaurants belong to a few major categories.
- Online ordering is highly popular.
- Ratings are positively influenced by customer votes.
- Dataset shows high positive skewness in some numerical columns.
- Outliers exist in pricing and vote distributions.
- Premium restaurants generally receive more votes.
